This is a simple solver for the **Wishful Memories** puzzle in **Infinity Nikki (Version 2.0)**.

## Usage

You can use the solver in two ways:

* **Download** the `index.html` file and open it in your browser
  **OR**
* **Visit the online version:** [https://krymzone.github.io/Infinity-Nikki-2.0-Tetris-Solver/](https://krymzone.github.io/Infinity-Nikki-2.0-Tetris-Solver/)

The grid is fully clickable, allowing you to set any default pattern you like, as shown below:

## Solution Types

**Solve (Least Blocks):**
  Attempts to solve the puzzle using the fewest pieces possible.

**Solve (Use Most Abundant) - Recommended:**
  Prioritizes using whichever pieces you have the most of. For example, if you have 15 S-shaped pieces and only 3 of every other piece, this option will try to use the S pieces first.
